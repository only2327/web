Basic HTML Tags

A reference table of simple and commonly used HTML tags.

No. Tag Syntax Output / Preview Description
1 <h1> to <h6> <h1>Heading</h1>

Heading

 Used to define HTML headings. <h1> is the largest.
2 <p> <p>Paragraph text</p>

Paragraph text

 Defines a paragraph of text.
3 <a> <a href="#">Link</a> Link Creates a hyperlink to other web pages or resources.
4 <ul> <ul>...</ul>
  • Item
 Defines an unordered (bulleted) list.
5 <ol> <ol>...</ol>
  1. Item
 Defines an ordered (numbered) list.
6 <li> <li>List item</li>
  • List item
    		•  Defines a list item inside an <ul> or <ol>.
    7 <img> <img src="image.jpg" alt="Img"> Embeds an image into the document.
    8 <br> <br> Line
    Break    		 Inserts a single line break.
    9 <hr> <hr>
    		Creates a thematic break (horizontal rule) between paragraph-level elements.
    10 <b> <b>Bold</b> Bold Makes text bold.
    11 <i> <i>Italic</i> Italic Makes text italicized.
    12 <sub> H<sub>2</sub>O H2O Displays text as subscript (below baseline).
    13 <sup> x<sup>2</sup> x2 Displays text as superscript (above baseline).
    14 <input> <input type="text"> Defines an input control for user data.
    15 <button> <button>Click Me</button> Defines a clickable button.
