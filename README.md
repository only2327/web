## 🛠 Prerequisites

Before running the backend or React projects, ensure you have the following installed on your machine:
- [Node.js](https://nodejs.org/) (v16 or higher)
- [MongoDB Community Server](https://www.mongodb.com/try/download/community) (must be running locally)
- A code editor like [VS Code](https://code.visualstudio.com/)

## 🛑 Troubleshooting Common Issues
- **MongooseError: Operation buffering timed out:** Your local MongoDB isn't running. Start MongoDB locally (e.g., via Windows Services or `mongod`).
- **EADDRINUSE: address already in use:** A server is already running on the same port. Stop previous instances (Ctrl+C) before running a new project.
- **Cannot find module 'X':** You missed the `npm install` step for that specific project. Make sure you run it inside the correct folder.
